- **数据与逻辑分离**：通过 `index.html` 按顺序加载 `data/cards.js`、`data/quiz.js` 和 `app.js`，利用全局变量 `window.CARDS` 和 `window.QUIZZES` 实现数据注入，避免异步请求。
- **单页应用 (SPA) 架构**：在 `index.html` 中预定义所有视图容器（首页、学习、测验、词库、我的），由 `app.js` 中的 `nav()` 函数通过切换 CSS 类 `.sc.on` 控制视图显示。
- **状态持久化**：核心学习进度（SRS 状态 `w3_r`）和统计信息（`w3_s`）直接序列化存储于 `localStorage`，在 `app.js` 初始化时同步加载。
- **视图渲染模式**：采用命令式 DOM 操作，根据当前状态动态生成 HTML 字符串并注入容器（如 `showCard`, `renderLib`），无虚拟 DOM 或框架依赖。
