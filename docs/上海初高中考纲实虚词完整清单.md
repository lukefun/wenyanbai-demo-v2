# 核心数量对照（单独区分）
1. 虚词
- 初中必学高频：6个
- 高中必学高频：16个
2. 实词
- 初中必学高频：150个（含固定双音节古今异义短语）
- 高中必学高频：300个（单字实词，完整包含初中全部150个单字实词）
# 关键说明
1. 上海本地规则：**考纲列出=试卷唯一高频词池**，不存在“必学一套、高频另一套”；
2. 高中300实词仅单字，初中150包含单字+18组双音节古今异义；
3. 高中16虚词完全覆盖初中6个虚词，新增10个；
4. 两者数量完全不相等，高中实虚词总量远多于初中。
