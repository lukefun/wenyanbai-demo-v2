# 第一部分：18 个高频虚词

---

## 一、之

### （一）代词

| 用法 | 说明 | 例句（出处） |
|------|------|-------------|
| 代人（第三人称） | 他/她/他们 | **使之然也**（《劝学》——代"木"） |
| 代事 | 这件事 | **均之二策**（《廉颇蔺相如列传》——代"予璧"与"不予璧"） |
| 代物 | 它 | **聚之咸阳**（《过秦论》——代"兵"） |
| 代人（活用为第一人称） | 我 | **不知将军宽之至此也**（《廉颇蔺相如列传》——"之"代廉颇自己） |

### （二）结构助词"的"

> 鹏**之**背，不知其几千里也 / 此非孟德**之**困于周郎者乎（《赤壁赋》）/ 古**之**学者必有师（《师说》）/ 人**之**相与（《兰亭集序》）

### （三）主谓之间，取消句子独立性（不译）

> 师道**之**不传也久矣（《师说》）/ 道**之**所存，师**之**所存（《师说》）/ 吾妻**之**死年（《项脊轩志》）/ 夫祸患常积于忽微，而智勇多困于所溺，岂独伶人也哉（→ 盛衰**之**理，虽曰天命，岂非人事哉）（《伶官传序》）/ 悟已往**之**不谏（《归去来兮辞》）

### （四）宾语前置的标志（不译）

> 句读**之**不知，惑**之**不解（《师说》）
> 何功**之**有哉？ / 何陋**之**有？

### （五）定语后置的标志（不译）

> 蚓无爪牙**之**利，筋骨**之**强（《劝学》）
> 居庙堂**之**高则忧其民

### （六）音节助词（不译）

> 顷**之**，持一象笏至（《项脊轩志》）/ 久**之**，能以足音辨人（《项脊轩志》）

### （七）动词：到……去、往

> 及其所**之**既倦（《兰亭集序》——"之"为动词"到达、追求"）/ 夜驰**之**沛公军（《鸿门宴》）/ 项伯乃夜驰**之**沛公军 / 杳不知其所**之**也（《阿房宫赋》——到……去）/ 而又有羡于长江之无穷——→ 不在此列

## 二、其

### （一）代词

| 用法 | 说明 | 例句 |
|------|------|------|
| 第三人称（作定语） | 他的/它的 | 弟子不必不如师，师不必贤于**其**弟子（《师说》）；**其**曲中规（《劝学》） |
| 第三人称（作主语） | 他/它 | **其**为惑也，终不解矣（《师说》）；**其**皆出于此乎（《师说》——表推测，兼语气） |
| 第一人称 | 我的/自己 | 而余亦悔**其**随之而不得极夫游之乐也（→ 类似用法："失**其**所与"《烛之武退秦师》） |
| 指示代词（远指） | 那/那些 | **其**后秦伐赵（《廉颇蔺相如列传》）；**其**制稍异于前（《项脊轩志》） |
| 指示代词（其中） | 其中的 | 于乱石间择**其**一二扣之（《石钟山记》） |

### （二）语气词

| 用法 | 说明 | 例句 |
|------|------|------|
| 表推测 | 大概/恐怕 | **其**皆出于此乎（《师说》） |
| 表反问 | 难道/怎么 | **其**可怪也欤（《师说》）；尽吾志也而不能至者……**其**孰能讥之乎 |
| 表祈使/期望 | 一定/还是/可要 | 尔**其**无忘乃父之志（《伶官传序》）；吾**其**还也（《烛之武退秦师》） |
| 表选择 | 是……还是…… | 天之苍苍，**其**正色邪？**其**远而无所至极邪？ |

## 三、以

### （一）介词

| 用法 | 说明 | 例句 |
|------|------|------|
| 表工具/方式 | 用/拿/把 | 乃奋臂**以**指拨眦（《左忠毅公逸事》）；**以**勇气闻于诸侯（《廉颇蔺相如列传》） |
| 表凭借/依据 | 凭/按照/根据 | 余**以**乾隆丁亥年 / 是**以**区区不能废远（《陈情表》）；苟**以**天下之大（《六国论》） |
| 表原因 | 因为/由于 | 臣**以**险衅（《陈情表》）；**以**其无礼于晋（《烛之武退秦师》）；不赂者**以**赂者丧（《六国论》）；赵王岂**以**一璧之故欺秦邪（《廉颇蔺相如列传》） |
| 表时间/处所 | 在/从 | 武**以**始元六年春 / 果得于数里外 |
| 表率领 | 率领 | 欲**以**客往赴秦军 |

### （二）连词

| 用法 | 说明 | 例句 |
|------|------|------|
| 表目的 | 来/用来/以便 | 作《师说》**以**贻之（《师说》）；还矢先王，而告**以**成功 / **以致**天下之士（《过秦论》） |
| 表结果 | 以致/从而 | 日削月割，**以**趋于亡（《六国论》） |
| 表并列 | 而且/又 | 夫夷**以**近，则游者众 |
| 表修饰 | ……地 | 拥火**以**入 / 余与四人拥火**以**入 |
| 表因果 | 因为 | 晋侯、秦伯围郑，**以**其无礼于晋（《烛之武退秦师》） |

### （三）动词：认为、以为

> 自**以**心为形役（《归去来兮辞》——"以心为形役"即"把心当作形体的奴仆"/"认为心被形体驱使"）
> 皆**以**美于徐公

### （四）固定搭配

| 短语 | 意义 | 例句 |
|------|------|------|
| 是以 | 因此 | **是以**区区不能废远（《陈情表》） |
| 以为 | 把……作为/认为 | 铸**以为**金人十二（《过秦论》）；自以为得之矣（《石钟山记》） |
| 以故 | 因此 | **以故**法为其国与此同 |
| 无以 | 没有用来……的 | 不积小流，**无以**成江海（《劝学》） |
| 有以 | 有用来……的 | 吾终当有**以**活汝 |
| 所以 | ……的原因/用来……的 | 师者，**所以**传道受业解惑也（《师说》）；**所以**游目骋怀（《兰亭集序》） |

## 四、而

### （一）连词

| 用法 | 说明 | 例句 |
|------|------|------|
| 表并列 | 又/和/并且 | 蟹六跪**而**二螯（《劝学》）；侣鱼虾**而**友麋鹿（《赤壁赋》） |
| 行顺承 | 就/然后/接着 | 觉今是**而**昨非（《归去来兮辞》）；秦人开关延敌，九国之师逡巡**而**不敢进（《过秦论》） |
| 行递进 | 而且 | 君子博学**而**日参省乎己（《劝学》） |
| 表转折 | 但是/却 | 青，取之于蓝，**而**青于蓝（《劝学》）；**而**刘病日笃（《陈情表》）；**而**听细说（《鸿门宴》）；惑**而**不从师（《师说》） |
| 表假设 | 如果 | 诸君**而**有意，瞻余马首可也 |
| 表修饰 | ……地/……着 | 吾尝跂**而**望矣（《劝学》）；顺流**而**东也（《过秦论》）；扣舷**而**歌之（《赤壁赋》）；弃甲曳兵**而**走 |
| 表因果 | 因而 | 其下圣人也亦远矣，**而**耻学于师（《师说》）——兼转折 |

### （二）代词：你（通"尔"）

> **而**翁归，自与汝复算耳

### （三）固定搭配

| 短语 | 意义 |
|------|------|
| 而已 | 罢了 |
| 而后 | 然后才 |
| 而况 | 何况 |
| 既而 | 不久 |

## 五、于

### 介词

| 用法 | 说明 | 例句 |
|------|------|------|
| 表处所 | 在/到/从 | 会**于**会稽山阴之兰亭（《兰亭集序》）；得双石**于**潭上（《石钟山记》）；辞楼下殿，辇来**于**秦（《阿房宫赋》） |
| 表对象 | 对/向/对于 | 寡人之**于**国也 / 学**于**余（《师说》）；耻学**于**师（《师说》） |
| 表比较 | 比 | 冰，水为之，**而寒于**水（《劝学》）；师不必贤**于**弟子（《师说》）；其势弱**于**秦（《六国论》） |
| 表被动 | 被 | 不拘**于**时（《师说》）；此非孟德之困**于**周郎者乎（《赤壁赋》）；而君幸**于**赵王（《廉颇蔺相如列传》） |
| 表原因 | 由于/因为 | 业精**于**勤，荒**于**嬉 |
| 表范围 | 在……方面 | 于其身也（《师说》——对于） |

### 固定搭配

| 短语 | 意义 | 例句 |
|------|------|------|
| 于是 | 在这种情况下/于是 | 吾祖死于是 / **于是**秦人拱手而取西河之外（《过秦论》） |
| 至于 | 到了……的时候/到 | **至于**颠覆，理固宜然（《六国论》） |
| 见…于 | 被 | 臣诚恐**见**欺**于**王而负赵 |

## 六、为

### （一）介词（wèi）

| 用法 | 说明 | 例句 |
|------|------|------|
| 表对象 | 替/给 | 臣请**为**大王言之；**为**击破沛公军（《鸿门宴》——替我） |
| 表目的 | 为了 | 慎勿**为**妇死 |
| 表原因 | 因为 | 盘庚不**为**怨者故改其度 |
| 表被动（wéi） | 被 | **为**天下笑者，何也（《过秦论》）；吾属今**为之**虏矣（《鸿门宴》）；身死人手，**为**天下笑（《过秦论》） |

### （二）动词（wéi）

| 意义 | 例句 |
|------|------|
| 做/担任/治理 | 使子婴**为**相（《鸿门宴》）；以**为**观游（《种树郭橐驼传》） |
| 成为/变成 | 冰，水**为**之（《劝学》）；斩木**为**兵（《过秦论》） |
| 是 | 如今人方**为**刀俎，我**为**鱼肉（《鸿门宴》） |
| 认为/以为 | 窃**为**大王不取也（《鸿门宴》） |

### （三）语气词（wéi，句末）

> 何辞**为**？（《鸿门宴》——呢）
> 何以为家？

## 十、者

### （一）代词

| 用法 | 说明 | 例句 |
|------|------|------|
| ……的人 | 指人 | 师**者**，所以传道受业解惑也（《师说》）；古之学**者**必有师（《师说》）；逝**者**如斯（《赤壁赋》）；有以文人画士孤癖之隐明告鬻梅**者**（《病梅馆记》） |
| ……的事/物 | 指事/物 | 虽有槁暴，不复挺**者**，輮使之然也（《劝学》） |
| ……的原因 | 表原因 | 吾妻死**之年**所手植也→ 类似结构：所以遣将守关**者**（《鸿门宴》） |

### （二）语气助词

| 用法 | 说明 | 例句 |
|------|------|------|
| 表停顿/提顿 | 不译 | 北山愚公**者**，年且九十 |
| 用在时间词后 | 不译 | 今**者**有小人之言（《鸿门宴》）；昔**者**…… / 顷**之** |
| 与"如""若"配合 | ……的样子/像……似的 | 言之，貌若甚戚**者** |

## 十一、所

### （一）"所" + 动词 = 名词性短语（……的）

| 说明 | 例句 |
|------|------|
| ……的人/事物/地方 | 问女何**所**思 / 为国者无使为积威之**所**劫哉（《六国论》）；得**所**谓石钟者（《石钟山记》）；不如其**所**之也（《阿房宫赋》） |

### （二）"为……所" 表被动

> **为**秦人积威之**所**劫（《六国论》）
> 若属皆且为**所**虏（《鸿门宴》）

### （三）"所以"

| 意义 | 说明 | 例句 |
|------|------|------|
| ……的原因 | 表原因 | 此世**所以**不传也（《石钟山记》） |
| 用来……的 | 表工具/方法 | 师者，**所以**传道受业解惑也（《师说》）；**所以**游目骋怀（《兰亭集序》） |

### （四）"所" + 数词 = 表约数

> 父去里**所**，复还（"里所"即"一里左右"）

## 十二、何

### （一）疑问代词

| 用法 | 说明 | 例句 |
|------|------|------|
| 什么 | 代事物 | 大王来**何**操（《鸿门宴》）；子**何**恃而往 |
| 哪里/哪里 | 代处所 | 豫章故郡→ 类：水哉，水哉，**何**取于水也 |
| 为什么 | 代原因 | **何**者？严大国之威以修敬也 / 此**何**地也（《左忠毅公逸事》） |
| 怎么 | 表反问 | **何**辞为（《鸿门宴》） |

### （二）固定搭配

| 短语 | 意义 | 例句 |
|------|------|------|
| 如何/奈何 | 怎么办 | 为之**奈何**（《鸿门宴》） |
| 何以 | 凭什么/怎么 | 君**何以**知燕王（《廉颇蔺相如列传》） |

| 何乃 | 为什么竟 | **何乃**太区区 |

## 十五、焉

| 用法 | 词性 | 说明 | 例句 |
|------|------|------|------|

# 附录：易混辨析速查表

| 辨析对象 | 区别要点 | 典型例句 |
|----------|---------|---------|
| **以**（介词）vs **以**（连词） | 介词后接名词/代词；连词连接动词/分句 | 以勇气闻（介词，凭）/ 作《师说》以贻之（连词，来） |
| **而**（转折）vs **而**（承接） | 转折：前后语意相反；承接：前后动作先后发生 | 青取之于蓝**而**青于蓝（转折）/ 觉今是**而**昨非（并列） |
| **其**（代词）vs **其**（语气词） | 代词可做主/宾/定语；语气词在句首/句末 | **其**曲中规（代词，它的）/ **其**皆出于此乎（语气词，大概） |
| **之**（代词）vs **之**（助词） | 代词可做宾语；助词"的"/取独 | 聚**之**咸阳（代词，它）/ 古**之**学者（助词，的） |
| **于**（在）vs **于**（比）vs **于**（被） | 看上下文语境 | 会**于**会稽（在）/ 冰寒**于**水（比）/ 不拘**于**时（被） |
| **为**（wéi 做）vs **为**（wèi 替） | wéi：动词/判断/被动；wèi：介词表目的/对象 | 使子婴**为**相（wéi，做）/ **为**击破沛公军（wèi，替） |
| **乃**（于是）vs **乃**（竟然）vs **乃**（是） | 顺承/出乎意料/判断 | **乃**还（于是）/ **乃**不知有汉（竟然）/ 当立者**乃**公子扶苏（是） |
| **因**（于是）vs **因**（趁着）vs **因**（沿袭） | 看语境 | **因**持璧却立（于是）/ **因**利乘便（趁着）/ **因**遗策（沿袭） |
